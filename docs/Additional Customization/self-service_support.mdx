# Self-Service Support

Self-service support lets cardholders directly submit inquiry tickets to Triple.
<figure>
    <Image img={require('./images/self-service_support_1.png')} alt="Enabling self-service support" style={{ marginTop: '20px', maxWidth: '600px'}}/>
    <figcaption style={{fontStyle: 'italic', fontSize: 'smaller', fontFamily: 'sans-serif', marginBottom: '50px'}}>Enabling self-service support</figcaption>
</figure>

All communication happens within the branded user interface. Cardholders can file tickets but cannot add comments or updates. 
To dispute a decision, a cardholder must open a new ticket.
