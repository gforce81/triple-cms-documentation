# Styles

Triple CMS allows you to configure:
- Primary Color
- Secondary Color
- Offer Title Color

The colors are applied to the various UI elements according to their purpose and position.

You can also define:
- Primary site font
- Secondary site font (future)

The primary font will be applied to titles and core elements while the secondary style font will be applied to longer texts.
