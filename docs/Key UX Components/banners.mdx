# Banners

Banner sections can be placed throughout a page, and you can add multiple sections per page. Banners serve several key roles:
- Promote specific content, deals, merchants, or pages
- Add clear CTAs for cardholders
- Provide visual diversity within a single page

Banners are creative assets that exist within the CMS—they are not deals themselves. Each banner includes a CTA (call-to-action), which can:
- Link to a specific category filter or category page (future)
- Link to an external website
- Link to a specific search query
- Link to a specific offer

To add a banner, go to the `Add Section` menu. You can choose from three banner components:
- Extra large banner that covers the entire width of the container
- Large banner that covers 75% of the container width and partially shows the next banner in the carousel
- Medium banners that can be placed side-by-side to cover the entire container width

After adding a banner section, you can configure each banner's creative assets and CTAs.
Each CTA type offers different customization options:
### Category Filter
- Select the category/categories for cardholder redirection
- Deal activation condition: include, exclude, or don't filter on activation requirements
- Add a link to the map view for local, in-person deals
- Offer Availability: online, in-person, or both
- Offer Type: Affiliate, Card-Linked, Merchant-Fulfilled (coupon codes at checkout)
- Exclude specific offers from results
- Add creative assets for desktop and mobile with alternate text
