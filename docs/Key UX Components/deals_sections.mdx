# Deals Sections

Deals are central to pages built with the Triple CMS and Triple White Label UI. These deals are typically organized into sections, each configured to show a specific set of offers for the cardholder. 
You can add as many deal sections as needed to a page.
When adding a deals section, you can choose from three layout options:
- Large cards (3 cards visible across a desktop screen)
- Medium cards (4 cards visible across a desktop screen)
- Small cards (6 cards visible across a desktop screen)

Mix and match these layout styles to create a visually engaging page.

You can refine your deal sections by using filters to create more targeted and engaging displays. 
Each section can be given a custom name (title) to clearly indicate its purpose and grouping for cardholders.
Here are the available filtering options:
- **Category:** Show deals from one or multiple specific categories.
- **Specific Deal IDs:** Display only deals identified by their unique IDs.
- **Exclude Deal IDs:** Prevent certain deals from appearing in the section.
- **Deal Type:** Filter by Affiliate, Card-Linked, or Merchant-Fulfilled (coupon code applied at checkout).
- **Deal Availability:** Show deals available In-Person or Online.
- **Activation Condition:** Determine if a prior purchase is required to activate the deal.
- **Link to Map:** Display a link to a map view for local deals.
