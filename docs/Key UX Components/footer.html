Footer

The footer is optional. Enable or disable the footer component by going to Settings > Show Footer

To configure the footer, select it in the configuration area and click the "edit" icon. You can customize the background and font colors, add a logo, include custom links, and specify copyright text to appear at the bottom of the page.
