# Footer

The footer is optional. Enable or disable the footer component by going to `Settings > Show Footer`
<figure>
    <Image img={require('./images/edit_footer_1.png')} alt="Enabling the footer component" style={{ marginTop: '20px', maxWidth: '600px'}}/>
    <figcaption style={{fontStyle: 'italic', fontSize: 'smaller', fontFamily: 'sans-serif', marginBottom: '50px'}}>Enabling the footer component</figcaption>
</figure>

To configure the footer, select it in the configuration area and click the "edit" icon.
You can customize the background and font colors, add a logo, include custom links, and specify copyright text to appear at the bottom of the page.

</figure>
