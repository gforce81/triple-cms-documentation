# Header

The header is always present as it contains essential navigation elements like Search and the pages menu for the White Label UI. 
To configure the header, select it in the configuration area and click the "edit" icon.
<figure>
    <Image img={require('./images/edit_header_1.png')} alt="Editing the header component" style={{ marginTop: '20px', maxWidth: '600px'}}/>
    <figcaption style={{fontStyle: 'italic', fontSize: 'smaller', fontFamily: 'sans-serif', marginBottom: '50px'}}>Editing the header component</figcaption>
</figure>

You can customize the background and font colors, along with the primary and secondary logos and an optional title.
