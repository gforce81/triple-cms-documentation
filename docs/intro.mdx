# Introduction

Triple CMS (Content Management System) is a tool developed by Triple that allows visual and simple creation of new display experiences for the Triple White Label UI.

The CMS enables publishers (financial institutions) to create new digital experiences by customizing how deals appear to cardholders.

At a high level, CMS handles:
<ul style={{ listStyleType: 'none', paddingLeft: '1em' }}>
  <li>✅ Organizing the visual presentation of deals</li>
  <li>✅ Creating and editing White Label UI versions</li>
  <li>✅ Creating and editing custom campaign pages</li>
  <li>✅ Creating and editing custom merchant pages</li>
</ul>

CMS does **not** handle:
<ul style={{ listStyleType: 'none', paddingLeft: '1em' }}>
  <li>❌ Deal creation, editing, or management</li>
  <li>❌ Publisher tag management for deals</li>
  <li>❌ Deal display status control</li>
  <li>❌ Recommendation and targeting functions</li>
</ul>
